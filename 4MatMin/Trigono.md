Persamaan trigono

sin --> ada kiri ada kanan
Sin, ebelah kanan juga sin
k --> konstanta x360 (sin,cos)
k --> konstant x180 (tan)
ex. k = 2 --> 720˚

jadi bikin x = x
## Sin
sin(2x+30˚) = sin(70) 0<x<360
1. 2x+30 = 70 + k.360
	1. 2x = 40 + k360
	2. x  = 20 +k180
2. 2x+30 = 110 + k.360
	1. x  = 40 +k180
3. HP = {20, 40, 200, 220}

## Cos
cos(2x+30˚) = cos(70) 0<x<360
1. 2x+30 = 70 + k.360
	1. 2x = 40 + k360
	2. x  = 20 +k180
2. 2x+30 = -70 + k.360
	1. x  = -100 +k180
3. HP = {80, 200, 260}

## Tan
tan(2x+30˚) = tan(70) 0<x<360
1. 2x+30 = 70 + k.180
	1. 2x = 40 + k00
	2. x  = 20 +k180
2. 2x+30 = 110 + k.180
	1. x  = 40 +k90
3. HP = {}
